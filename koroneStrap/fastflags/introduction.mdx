<Note>
    **FastFlags** are internal, configurable settings that allow Korone developers to remotely enable, disable, or adjust features and engine behavior without requiring a full software update.
    Users can modify these flags to unlock hidden engine settings, potentially improving performance or altering game features.
</Note>
With koroneStrap, you can apply FastFlags to Korone!

## Accessing FastFlags Configuration

To configure FastFlags, select option **5** (Set FastFlags) from the koroneStrap main menu. This will open the FastFlags Configuration interface.

## Managing FastFlags

The FastFlags Configuration menu provides several options for managing your FastFlags:

### 1. Add FastFlag
Add a new FastFlag to your configuration. You'll need to specify:
- The FastFlag name
- The FastFlag value

### 2. Remove FastFlag
Remove an existing FastFlag from your configuration.

### 3. Clear all FastFlags
Remove all configured FastFlags at once. Use this option to reset your FastFlags configuration.

### 4. Apply FastFlags
Apply your configured FastFlags to Korone. This writes your FastFlags configuration and makes it active.

### 5. Import FastFlags from JSON
Import a FastFlags configuration from a JSON file. This is useful for:
- Applying pre-configured FastFlag sets
- Sharing configurations between installations
- Backing up and restoring your FastFlags

### 0. Back to main menu
Return to the koroneStrap main menu by making changes or without changes.
