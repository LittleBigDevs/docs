# Common / Known Issues
## "Could not find executable. Error code: EXECNFOUND"
You forgot to download the bootstrapper! To do so, choose **Download/Update Bootstrapper** (option 6)

## "URI handling is only supported on Linux"
You got this error because you are using macOS or Windows.

## "Error reading fastFlags.json - invalid JSON format"
Check your JSON file formatting and check for errors.
